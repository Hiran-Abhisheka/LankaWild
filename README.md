## 📖 Overview

**LankaWild** is a modern, visually immersive web application showcasing the biodiversity of Sri Lanka through a cinematic web experience.

Built with a focus on **animations, performance, and UI/UX**, this project demonstrates advanced frontend development skills.

---

## ✨ Features

- 🌿 Wildlife exploration interface  
- 🎬 Smooth scroll animations & transitions  
- ⚡ Fast performance & optimized rendering  
- 📱 Fully responsive design  
- 🎨 Clean and modern UI  
- 🔥 Interactive effects  

---

## 🛠️ Tech Stack

- React.js  
- JavaScript (ES6+)  
- CSS3  
- Vercel
